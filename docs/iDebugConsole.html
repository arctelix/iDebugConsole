iDebugConsole

This package provides debugging and profiling tools. The primary purpose is to provide a means for controlling console output globally and within closures as required. An optional output window is provided as well. This is helpful for devices without a console window or when you do not require the full feature set of the native browser consoles. Finally a handy profiler for profiling javascript code execution and individual components. All that utility at just 0.309kb minified in a production environment and under 12kb minified in development.

An instance of the Debugger called iDebugger and the global debug method called debug are available after loading iDebugConsole.js. Use the debug method exactly how you would use console except that debug() may be called directly as a shortcut to debug.log(). Each Debugger instance provides independent control over the closure it is assigned to.
